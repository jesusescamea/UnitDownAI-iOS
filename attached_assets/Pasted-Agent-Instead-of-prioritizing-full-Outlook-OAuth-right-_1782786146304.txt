Agent,

Instead of prioritizing full Outlook OAuth right now, build an easier production-ready MVP: Speak Your Schedule.

Goal:
A technician should be able to verbally enter their daily schedule and have UnitDown convert it into draft jobs.

Example spoken input:
"Tomorrow I have Lowe's job 2606-37730 at 7 AM in Lincoln. Vendor meet at 10 AM. Contact is 510-472-1230. Then Best Buy job 2606-07588 at 7:30 in North Sac. PM survey."

UnitDown should extract structured jobs from the spoken schedule.

Do not change:
- Bundle ID
- Clerk auth
- Apple IAP
- Stripe
- version/build number
- existing job mode
- existing equipment records

==================================================
FEATURE NAME
==================================================

Add:

Speak Schedule

This should be available from:
- Dashboard quick actions
- Dispatch Inbox
- Add Job / Import Schedule screen

==================================================
WORKFLOW
==================================================

1. User taps Speak Schedule.
2. Voice recorder opens.
3. User speaks their schedule.
4. App transcribes speech to text.
5. AI parses the transcript into draft jobs.
6. User reviews extracted jobs.
7. User edits any missing fields.
8. User accepts selected jobs.
9. Jobs appear in:
   - Dashboard calendar
   - Today's Jobs
   - /job list
   - Future schedule date

==================================================
TRANSCRIPTION
==================================================

Use browser/device speech recognition if available.

If speech recognition is unavailable:
Show fallback:
"Voice schedule entry is unavailable on this device. Paste your schedule instead."

Offer:
- Paste Schedule
- Manual Job
- Import Calendar File

==================================================
AI PARSING
==================================================

Create one parsing function that accepts raw text and returns draft jobs.

Input:
raw transcript text

Output:
array of draft jobs

Each draft job should include:
- jobNumber
- customerName
- siteName
- location
- scheduledDate
- scheduledStartTime
- scheduledEndTime or timeWindow
- jobType
- priority
- phoneNumber
- contactName
- complaintOrTask
- notes
- source = "voice"
- needsReview = true
